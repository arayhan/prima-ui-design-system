# UI kit — arayhan.dev website

Interactive recreation of the arayhan personal site. Built from the owner's brief (no live site exists yet) — treat this as the reference design.

- `index.html` — interactive shell: theme toggle (system default + persisted), scroll progress, ⌘K command palette, view switching
- `data.js` — all dummy content (experience, projects, talks, blog posts, now/uses)
- `Shared.jsx` — Nav, Footer, CommandPalette
- `HomeScreen.jsx` — hero (open-to-work + positioning statement), about, experience tabs, case-study projects, speaking, now, contact
- `BlogScreens.jsx` — blog list + detail (dummy data, reading time, sticky TOC)
- `UsesScreen.jsx` — /uses gear & tools

Views: home, blog, post, uses — switch via nav or ⌘K.
